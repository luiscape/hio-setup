## H.io Setup
Scripts to setup [Humanitarian.io](http://humanitarian.io).

[![Build Status](https://travis-ci.org/luiscape/hio-setup.svg?branch=master)](https://travis-ci.org/luiscape/hio-setup)

## Usage
If you are on an `unix` system, run:
```shell
$ ./setup.sh
```

## Tests
To run test, do:
```shell
$ ./test.sh
```
